> ## What is DIAGNOSTIX?
>
> DIAGNOSTIX is an AI-powered OBD2 vehicle diagnostics app. It reads real-time vehicle data through the OBDLink MX+ adapter, runs AI analysis via Gemini, and delivers plain-English repair guidance — built for mechanics, not enthusiasts.
>
> **Billy has trades + construction background. This is domain knowledge most devs don't have.**
